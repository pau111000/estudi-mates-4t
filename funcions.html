🤔 Que es una funcion?

Imagina una maquina: tu le metes un numero (llamado x) y la maquina te devuelve otro numero (llamado y).

Ejemplo de la vida real: Un termometro es como una funcion. Le "metes" una hora del dia (x = las 3 de la tarde) y te "devuelve" una temperatura (y = 28 grados). A cada hora le corresponde UNA temperatura.

Una funcion es una relacion que a cada valor de x le asigna un unico valor de y. Se escribe asi: y = f(x), que se lee "y igual a f de x".

Cuando dibujamos todos esos puntos (x, y) en unos ejes, obtenemos una grafica. Y de esa grafica podemos sacar un monton de informacion!


Cuando veas f(x) piensa: "le meto x a la maquina y me sale esto". Por ejemplo, si f(x) = x + 3, y meto x = 2, sale f(2) = 2 + 3 = 5.

📋 Que vamos a estudiar de cada funcion?

Cuando nos dan la grafica de una funcion, tenemos que analizar estas cosas:

1. Dominio → En que valores de x existe la funcion (donde "vive" la grafica)
2. Recorrido → Que valores de y toma la funcion (hasta donde llega de alto/bajo)
3. Cortes con ejes → Donde la grafica toca el eje X y el eje Y
4. Continuidad → Si la grafica se puede dibujar sin levantar el lapiz
5. Crecimiento/Decrecimiento → Donde sube y donde baja
6. Maximos y Minimos → Los puntos mas altos (cimas) y mas bajos (valles)

🎯 Dominio - Donde "vive" la funcion


El dominio son todos los valores de x donde la funcion existe y tiene sentido. Dicho facil: si miras la grafica de izquierda a derecha, el dominio es todo el trozo del eje X que la grafica cubre.
Piensalo asi: Imagina que la grafica es un camino. El dominio es la parte del suelo (eje X) por donde pasa ese camino. Si hay un hueco en el camino, ese punto NO esta en el dominio.

📐 Polinomios (x², x³, 2x + 3...)

Los polinomios son las funciones mas simples. Su dominio es SIEMPRE todos los numeros reales. No hay ninguna x que les cause problemas.

Si f(x) es un polinomio → Dominio = todos los reales = (-∞, +∞)

f(x) = x² + 3x - 5
Como es un polinomio (solo tiene sumas, restas y potencias de x), puedes meter CUALQUIER numero. Dom = (-∞, +∞)

√ Raices cuadradas

Aqui hay una regla importante: lo que hay dentro de la raiz NO puede ser negativo (no puedes sacar la raiz cuadrada de un numero negativo).

f(x) = √g(x) → Necesitas que g(x) ≥ 0

f(x) = √(x - 4)

1
Lo de dentro de la raiz es: x - 4
2
Tiene que ser ≥ 0, asi que: x - 4 ≥ 0
3
Resolvemos: x ≥ 4
4
Dominio = [4, +∞) (desde 4 en adelante, incluyendo el 4)

Usamos [ ] cuando el numero SI entra y ( ) cuando NO entra. Como x = 4 si vale (√0 = 0), ponemos [4. El infinito SIEMPRE lleva parentesis ().

➗ Fracciones (funciones racionales)

Regla de oro: el denominador (lo de abajo) NO puede ser cero. Division entre cero = prohibido.

f(x) = algo / Q(x) → Necesitas que Q(x) ≠ 0

f(x) = 1 / (x - 3)

1
El denominador es: x - 3
2
NO puede ser 0, asi que: x - 3 ≠ 0
3
Resolvemos: x ≠ 3
4
Dominio = Todos los reales excepto 3 = (-∞, 3) ∪ (3, +∞)

f(x) = (2x+6) / (x² - 3x + 2)

1
El denominador es: x² - 3x + 2
2
Lo igualamos a 0: x² - 3x + 2 = 0
3
Factorizamos: (x - 1)(x - 2) = 0 → x = 1 o x = 2
4
Dominio = Todos los reales excepto 1 y 2 = (-∞, 1) ∪ (1, 2) ∪ (2, +∞)

Receta rapida para fracciones: 1) Coges el denominador. 2) Lo igualas a cero. 3) Resuelves. 4) Esos valores son los que EXCLUYES del dominio.

〰️ Continuidad


Una funcion es continua si puedes dibujar su grafica sin levantar el lapiz del papel. Asi de simple.
Piensalo asi: Imagina que vas en bici por un camino. Si el camino no tiene baches, huecos ni saltos, es continuo. Si hay un agujero o un salto, ahi hay una discontinuidad.
Buena noticia: Los polinomios (x², 2x+3, x³-x...) son SIEMPRE continuos en todos los reales. Nunca tienen problemas.

⚡ Tipos de discontinuidades

Cuando la grafica tiene algun "problema" y tienes que levantar el lapiz, hay 3 tipos:

💥 1. Salto finito

La grafica va por un lado y de repente "salta" a otro nivel, como si subieras un escalon.

Como se ve: Dos trozos de curva a alturas diferentes. En el punto del salto ves un circulo abierto (hueco) arriba y uno cerrado (lleno) abajo, o al reves.
y 4| ● 3| 2| ──────○ 1| +──────────── x salto!

🚀 2. Salto infinito (asintota vertical)

La grafica se "dispara" hacia arriba o hacia abajo sin limite. Se va al infinito. Hay una linea vertical invisible que la funcion nunca toca.

Como se ve: La curva sube (o baja) cada vez mas rapido, acercandose a una linea vertical pero sin tocarla nunca.
y | | | | / | | / | |/ -----+------- x /| / | / | asintota | vertical

Las funciones tipo 1/(x-a) siempre tienen una asintota vertical en x = a. El denominador se hace cero ahi y la funcion "explota".

🕳️ 3. Evitable (un agujero)

Hay un "agujerito" en la grafica. La curva pasa suavemente, pero falta justo ese punto. Si lo rellenaras, la funcion seria continua.

Como se ve: Un circulito abierto (hueco) en la curva, pero la curva sigue su camino normal por los dos lados.
y | / | ○ ← agujerito | / |/ +──────── x

📈 Crecimiento y Decrecimiento


Una funcion crece en un intervalo si, al caminar hacia la derecha, la grafica sube. Y decrece si al caminar hacia la derecha, la grafica baja.
Piensalo asi: Imagina que caminas por una montana de izquierda a derecha. Si vas cuesta arriba, la funcion crece. Si vas cuesta abajo, decrece.
y CRECE DECRECE CRECE | /\ | / \ / | / \ / | / \ / | / \ / | / \ / | / V ← minimo +───────────────────── x ↑ minimo

Regla facil: Mira la grafica de IZQUIERDA a DERECHA. Si sube → crece. Si baja → decrece. Siempre leemos de izquierda a derecha, como un libro.

⛰️ Maximos y Minimos


Un maximo es un punto donde la funcion deja de subir y empieza a bajar (la cima de una montaña). Un minimo es un punto donde deja de bajar y empieza a subir (el fondo de un valle).
Maximo = cima de montaña: primero subes (crece) y luego bajas (decrece). 📈→⛰️→📉
Minimo = fondo de valle: primero bajas (decrece) y luego subes (crece). 📉→🏜️→📈

f(x) = -x² (parabola boca abajo)

La funcion sube hasta x = 0 y luego baja. Entonces:
- Tiene un MAXIMO en (0, 0) (es la cima)
- Crece en (-∞, 0) y decrece en (0, +∞)

Para recordar: Maximo = crece y LUEGO decrece (↗↘). Minimo = decrece y LUEGO crece (↘↗).

📏 Recorrido (o Rango)


El recorrido son todos los valores de y que la funcion puede tomar. Si el dominio mira el eje X, el recorrido mira el eje Y.
Piensalo asi: Pon una regla vertical al lado de la grafica. El recorrido son todas las alturas (valores de y) que la grafica alcanza.

f(x) = x² (parabola boca arriba)

La grafica tiene su punto mas bajo en y = 0 (el vertice) y sube infinitamente. Nunca baja de 0.
Recorrido = [0, +∞) → y puede ser 0 o cualquier numero positivo.

f(x) = -x² (parabola boca abajo)

La grafica tiene su punto mas alto en y = 0 y baja infinitamente. Nunca sube de 0.
Recorrido = (-∞, 0] → y puede ser 0 o cualquier numero negativo.

✂️ Cortes con los ejes


Los cortes con los ejes son los puntos donde la grafica toca o cruza el eje X y el eje Y.

Corte con el eje X (donde y = 0)

Para encontrarlo: haces f(x) = 0 y resuelves. Los puntos tendran forma (a, 0).


f(x) = x² - 4

1
Hacemos f(x) = 0 → x² - 4 = 0
2
x² = 4 → x = ±2
3
Cortes con eje X: (2, 0) y (-2, 0)

Corte con el eje Y (donde x = 0)

Para encontrarlo: calculas f(0). Solo puede haber UN punto, con forma (0, b).


f(x) = 3x² + 2x - 7

1
Calculamos f(0) = 3(0)² + 2(0) - 7 = -7
2
Corte con eje Y: (0, -7)

Eje X → y = 0 (resuelves). Eje Y → x = 0 (sustituyes). Solo hay que recordar eso!

📏 Notacion de intervalos

Usamos intervalos para decir "desde aqui hasta aqui". Hay que saber cuando usar parentesis y cuando corchetes:

[ ] = INCLUIDO (el numero SI entra). Circulo lleno ●
Se usa cuando la funcion SI vale en ese punto.
( ) = EXCLUIDO (el numero NO entra). Circulo vacio ○
Se usa cuando la funcion NO vale en ese punto, o para ∞ y -∞.
∞ y -∞ SIEMPRE van con parentesis ( ) porque no son numeros reales

(2, 5) = los numeros entre 2 y 5, SIN incluir ni el 2 ni el 5
[2, 5] = los numeros entre 2 y 5, SI incluidos el 2 y el 5
[2, 5) = incluye el 2 pero NO el 5
(-∞, 3) = todos los numeros menores que 3
[0, +∞) = el cero y todos los positivos
El simbolo ∪ (union) une dos intervalos. Por ejemplo: (-∞, 2) ∪ (2, +∞) significa "todos los reales excepto el 2".
